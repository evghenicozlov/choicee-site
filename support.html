Support

Need help with Choicee? We’re here for you.

📩 Contact Us

If you have any questions, issues, or feedback, please reach out:

Email: feedback.choicee.app@gmail.com
We usually respond within 24–48 hours.

🛠 Common Issues

App is not working / crashing

Make sure you’re using the latest version of Choicee
Restart the app or your device
Reinstall if the issue persists

Sound or haptics not working

Check device silent mode
Enable vibration in system settings
Ensure app permissions are granted

Spinner / coin animation lag

Close background apps
Lower system load
Update to latest iOS version

🔐 Account & Data

Choicee does not require login for core features. If any data is stored locally, it stays on your device unless explicitly shared.

💡 Feature Requests

We love feedback. If you have ideas for improvements or new features, email us anytime at:
feedback.choicee.app@gmail.com

🧾 Bug Reports (recommended format)

When reporting a bug, please include:

Device model
iOS version
Steps to reproduce the issue
Screenshots or screen recording (if possible)
